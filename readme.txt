== Description ==
The **RT Posts Contributors plugin** allows easy and quick way to assign contributors to WordPress posts.

Simple and flexible, RT Posts Contributors works with any WordPress theme. 
Easy to setup. 

	
 = Free VERSION (Core) Features =
    1. Admin can assign any author role users to posts as contributors. 
	
 = Display Features =
    1. Post page displays the contributors list.
    
== Installation ==

Manual Installation

1. Upload the `posts-contributors` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Visit Posts -> Edit/New Posts -> contributor.

4. After you have assigned the contributors to the posts, save the post.

5. Visit single page of the post or visit in post front page, you will see the contributor box.
